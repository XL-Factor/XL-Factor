How to Use This Tool

1. Click "Choose JSON File" to select a JSON file or paste JSON directly

2. Click "Conversion Options" to customize Excel output format

3. Click "Convert to Excel" to start the conversion process

4. Preview the converted data in the preview section

5. Click "Download Excel" to save the file

6. Use "Reset" to clear all inputs and start over

What JSON formats are supported?+
We support standard JSON formats including arrays of objects (most common) and single objects. The JSON should be properly formatted.
Is there a file size limit?+
The current limit is 5MB for file uploads. For larger files, consider pasting the JSON directly.
What Excel formats are available?+
You can choose between modern XLSX format (Excel 2007+), legacy XLS format (Excel 97-2003), or CSV (comma-separated values).
How are nested objects handled?+
You can choose to flatten nested objects (using dot notation like "address.city") or skip them entirely in the output.
