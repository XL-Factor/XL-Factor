How to Use This Tool

1. Select the type of conversion from the dropdown

2. Enter the value you want to convert

3. Select the unit you're converting from

4. Select the unit you're converting to

5. Click "Convert" to see the result

6. All conversions are saved in the history section

7. Click "Download as Excel" to save your conversion history

8. Use "Clear" to reset the current conversion

9. Use "Reset All" to clear everything and start fresh

What's the difference between Clear and Reset All?+
"Clear" resets only the current conversion inputs, while "Reset All" clears both the current inputs and the entire history.
How many conversions are saved?+
The converter saves up to 50 recent conversions in your session. When you download as Excel, all conversions are included.
Are all conversions precise?+
Conversions use standard conversion factors and are precise to 10 decimal places. Some conversions (like temperature) use exact formulas rather than simple multiplication.
Can I edit the Excel file?+
Yes! The downloaded Excel file is fully editable. You can add more conversions, format cells, or add notes.
Why don't I see my specific unit?+
We've included the most common units for each category. If you need a specific unit added, please contact us with your suggestion.
