How to Use This Tool

1. Upload your main Excel file (the file you want to add data to)

2. Upload your lookup Excel file (the file containing the data to look up)

3. Click "VLOOKUP Options" to configure the VLOOKUP parameters

4. Select the sheets and columns to use for the VLOOKUP

5. Click "Process VLOOKUP" to apply the VLOOKUP formula

6. Click "Download Result File" to save the modified file

7. Use "Reset" to clear all files and start over

What Excel formats are supported?+
We support both modern XLSX format (Excel 2007+) and legacy XLS format (Excel 97-2003).
Is there a file size limit?+
The current limit is 10MB per file. For larger files, consider simplifying your workbook.
How does the VLOOKUP work?+
The tool matches values from your main file to the lookup file and returns the specified column's data, just like Excel's VLOOKUP function.
What if my data doesn't match exactly?+
Uncheck "Exact Match" for approximate matching, but exact matching is recommended for most cases.
