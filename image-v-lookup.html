How to Use This Tool

1. Upload your Excel file containing image names

2. Select a folder containing your images

3. Click "Image Lookup Options" to configure the import

4. Select the sheet and columns to use for the image lookup

5. Set the desired image dimensions

6. Click "Process Images" to import images into Excel

7. Click "Download Result File" to save the modified file

8. Use "Reset" to clear all files and start over

What Excel formats are supported?+
We support both modern XLSX format (Excel 2007+) and legacy XLS format (Excel 97-2003).
Is there a file size limit?+
The current limit is 10MB per file. For larger files, consider simplifying your workbook.
What image formats are supported?+
We support JPG, PNG, GIF, and BMP formats. You can select the specific format you're using.
How are images positioned in cells?+
Images are automatically sized to fit the specified dimensions and positioned to align with the corresponding cell.
