How to Use This Tool

1. Click "Choose Excel Files" to select one or more Excel (.xlsx or .xls) files

2. Click "Process" to start the conversion process

3. Each sheet in your Excel files will be converted to a separate CSV file

4. Once conversion is complete, click "Download ZIP" to save all CSV files

5. Use "Reset" to clear all files and start over

What Excel formats are supported?+
We support both .xlsx (Excel 2007+) and .xls (Excel 97-2003) formats.
How are multiple sheets handled?+
Each sheet in your Excel file will be converted to a separate CSV file. The files will be named with the original filename plus the sheet name.
Is there a file size limit?+
The current limit is 10MB per file. For larger files, consider splitting your data.
