How to Use This Tool

1. Click "Choose Excel Files" to select one or more Excel (.xlsx or .xls) files

2. Click "Convert to CSV" to start the conversion process

3. Once conversion is complete, click "Download ZIP" to save all CSV files and a merged CSV file

4. Use "Reset" to clear all files and start over

What Excel formats are supported?+
We support both .xlsx (Excel 2007+) and .xls (Excel 97-2003) formats.
Is there a file size limit?+
The current limit is 10MB per file. For larger files, consider splitting your data.
Will my files be merged?+
Yes, the tool creates individual CSV files for each Excel file and also merges all data into one combined CSV file.
